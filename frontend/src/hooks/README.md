# Custom React Hooks

This directory contains custom React hooks:

- `useWebSocket.js` - WebSocket connection management
- `useTheme.js` - Dark/light theme state and toggle
- `useLocalStorage.js` - localStorage synchronization
- `useKeyboardShortcuts.js` - Ctrl+Shift, Ctrl+Click tracking
- `useStatusColors.js` - Status color map management
- `useAssigneeSelection.js` - Selected assignee from navbar
- `useWorkbook.js` - Workbook units management
- `useSearch.js` - Global search with debounce
- `useBTCPrice.js` - Bitcoin price fetching
- `useHolidays.js` - Holiday calendar for business days
