# Page Components

This directory contains top-level page components:

- `LoginPage.jsx` - Authentication page
- `NotFoundPage.jsx` - 404 error page
- `OrdersListPage.jsx` - Main orders listing
- `OrderDetailsPage.jsx` - Order details with units table and analytics
- `RepairUnitPage.jsx` - Repair unit details with events log
- `SearchResultsPage.jsx` - Search results display
- `SettingsPage.jsx` - Settings/admin page
- `StatsPage.jsx` - Statistics dashboard
- `ProfitCalculatorPage.jsx` - Profitability calculator
