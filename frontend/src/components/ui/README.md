# Shared UI Components

This directory contains reusable UI components:

- `ElementBlock.jsx` - Content wrapper with border/shadow
- `LoadingSpinner.jsx` - Loading state indicator
- `ErrorMessage.jsx` - Error display component
- `DeleteButton.jsx` - Trash icon button (Ctrl+Shift visibility)
- `SearchBox.jsx` - Input + button combo for adding entities
- Form inputs, buttons, and other atomic UI elements
