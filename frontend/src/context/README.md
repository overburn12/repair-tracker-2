# React Context Providers

This directory contains React context providers for global state:

- `ThemeContext.jsx` - Theme (dark/light) state management
- `WebSocketContext.jsx` - WebSocket connection and subscriptions
- `WorkbookContext.jsx` - Workbook units list management
- `AssigneeContext.jsx` - Selected assignee management
- `AppDataContext.jsx` - Global app data (statuses, assignees, models)
