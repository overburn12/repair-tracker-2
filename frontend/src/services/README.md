# API Service Layer

This directory contains API service modules:

- `api.js` - Base API configuration and error handling
- `ordersService.js` - Repair orders CRUD operations
- `unitsService.js` - Repair units CRUD operations
- `searchService.js` - Search functionality
- `settingsService.js` - Assignees, statuses, models CRUD
- `statsService.js` - Statistics and analytics data
- `websocketService.js` - WebSocket connection and message handling
